A Fistful of Memes: An Act of Resilience

This is a website for the MAD project "MEMES OUT: How Filipinos Use New Media as Visual Language to Cope with Corruption". The 3D model of the fist was created by me using Autodesk Maya, which was further processed to Blender as a different filetype, and the website was built using Visual Studio Code.

The purpose of this project is to showcase the Filipinos' resilience to government corruption through the use of memes, as well as to create a fun and engaging experience for visitors.

I chose the name "A Fistful of Memes" because I wanted to create a website that is both visually striking and humorous, and I thought that the combination of a fist and memes would be a perfect fit.
